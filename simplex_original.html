Problema del Fabricante de Tres Modelos

Solucion por el Metodo Simplex — Variables Originales con Big-M

1Datos del Problema

Un fabricante produce tres modelos I, II y III de un producto utilizando materias primas A y B. Se desea maximizar los ingresos sujeto a las restricciones de materiales, fuerza de trabajo, demanda minima y proporciones de mercado.

Materia Prima Requerimientos por unidad Disponibilidad
  Modelo I (x1) Modelo II (x2) Modelo III (x3)  
A2354 000
B4276 000
Demanda minima (und.)200200150—
Precio por unidad ($)302050—
Datos adicionales: Las horas de trabajo por unidad del Modelo I son el doble que las del II y el triple que las del III. La capacidad total equivale a 1 500 unidades del Modelo I. El mercado exige producir en proporciones 3 : 2 : 5 para los modelos I, II y III.

2Formulacion Matematica

Variables de decision

x1 = unidades del Modelo I  |  x2 = unidades del Modelo II  |  x3 = unidades del Modelo III

Funcion objetivo

Maximizar   Z = 30x1 + 20x2 + 50x3

Restricciones

(1) 2x1 + 3x2 + 5x3 ≤ 4 000    (Materia Prima A)
(2) 4x1 + 2x2 + 7x3 ≤ 6 000    (Materia Prima B)
(3) 6x1 + 3x2 + 2x3 ≤ 9 000    (Fuerza de trabajo)
(4) 2x1 − 3x2 = 0    (Proporcion I : II = 3 : 2)
(5) 5x2 − 2x3 = 0    (Proporcion II : III = 2 : 5)
(6) x1 ≥ 200,   x2 ≥ 200,   x3 ≥ 150    (Demanda minima)
(7) x1, x2, x3 ≥ 0

Restriccion (3): horas por unidad: h, h/2, h/3. Multiplicando h·x1 + (h/2)x2 + (h/3)x3 ≤ 1500h por 6/h se obtiene 6x1 + 3x2 + 2x3 ≤ 9000.

3Cambio de Variable para Cotas Inferiores

Las restricciones de demanda minima (6) imponen que x1 ≥ 200, x2 ≥ 200, x3 ≥ 150. El Simplex exige variables no negativas que partan desde cero. Se introduce el cambio:

x'1 = x1 − 200  ≥  0   ⇒   x1 = x'1 + 200
x'2 = x2 − 200  ≥  0   ⇒   x2 = x'2 + 200
x'3 = x3 − 150  ≥  0   ⇒   x3 = x'3 + 150

Nueva funcion objetivo

Z = 30(x'1+200) + 20(x'2+200) + 50(x'3+150)
Z = 30x'1 + 20x'2 + 50x'3 + 6 000 + 4 000 + 7 500
Z = 30x'1 + 20x'2 + 50x'3 + 17 500
Equivalente a maximizar Z' = 30x'1 + 20x'2 + 50x'3 y sumar 17 500 al final.

Restricciones transformadas

(1') 2(x'1+200) + 3(x'2+200) + 5(x'3+150) ≤ 4 000
2x'1 + 3x'2 + 5x'3 + 400 + 600 + 750 ≤ 4 000  ⇒  2x'1 + 3x'2 + 5x'3 ≤ 2 250
(2') 4(x'1+200) + 2(x'2+200) + 7(x'3+150) ≤ 6 000
4x'1 + 2x'2 + 7x'3 + 800 + 400 + 1 050 ≤ 6 000  ⇒  4x'1 + 2x'2 + 7x'3 ≤ 3 750
(3') 6(x'1+200) + 3(x'2+200) + 2(x'3+150) ≤ 9 000
6x'1 + 3x'2 + 2x'3 + 1200 + 600 + 300 ≤ 9 000  ⇒  6x'1 + 3x'2 + 2x'3 ≤ 6 900
(4') 2(x'1+200) − 3(x'2+200) = 0  ⇒  2x'1 − 3x'2 + 400 − 600 = 0  ⇒  2x'1 − 3x'2 = 200
(5') 5(x'2+200) − 2(x'3+150) = 0  ⇒  5x'2 − 2x'3 + 1000 − 300 = 0  ⇒  −5x'2 + 2x'3 = −700
Equivalente a: 2x'3 − 5x'2 = 700   (multiplicado por −1 si hubiera RHS negativo)
Nota sobre la restriccion (5'): El RHS original es −700. Para mantenerlo positivo en el tableau se reescribe como −5x'2 + 2x'3 = 700 (se multiplico ambos lados por −1), lo que invierte los signos de los coeficientes pero no cambia la solucion.

4Forma Estandar — Metodo Big-M

Las restricciones (1'), (2'), (3') son desigualdades ≤: se agregan variables de holgura s1, s2, s3 ≥ 0.
Las restricciones (4') y (5') son igualdades: se agregan variables artificiales a1, a2 ≥ 0 con penalizacion −M (M muy grande) en el objetivo.

Maximizar   Z' = 30x'1 + 20x'2 + 50x'3 − M·a1 − M·a2

(R1) 2x'1 + 3x'2 + 5x'3 + s1 = 2 250
(R2) 4x'1 + 2x'2 + 7x'3 + s2 = 3 750
(R3) 6x'1 + 3x'2 + 2x'3 + s3 = 6 900
(R4) 2x'1 − 3x'2 + a1 = 200
(R5) −5x'2 + 2x'3 + a2 = 700
  x'1, x'2, x'3, s1, s2, s3, a1, a2 ≥ 0

Solucion basica factible inicial:   x'1 = x'2 = x'3 = 0,   s1=2250,   s2=3750,   s3=6900,   a1=200,   a2=700,   Z'=0.

Fila Z inicial ajustada (Big-M)

Como a1 y a2 estan en la base inicial con costo −M, se ajusta la fila Z sumando M×R4 y M×R5:

Fila Z inicial: [−30, −20, −50, 0, 0, 0, +M, +M | 0]
+ M × R4: [2M, −3M, 0, 0, 0, 0, M, 0 | 200M]
+ M × R5: [0, −5M, 2M, 0, 0, 0, 0, M | 700M]
= Fila Z ajust.: [−30−2M, −20+8M, −50−2M, 0, 0, 0, 0, 0 | −900M]
Criterio de entrada: el valor mas negativo indica la variable que entra.
Para M grande: −50−2M < −30−2M < 0  ⇒  x'3 entra primero

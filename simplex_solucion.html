Problema del Fabricante de Tres Modelos

Solucion por el Metodo Simplex

1Datos del Problema

Un fabricante produce tres modelos I, II y III de un producto utilizando materias primas A y B. Se desea maximizar los ingresos sujeto a las restricciones de materiales, fuerza de trabajo, demanda minima y proporciones de mercado.

Materia Prima Requerimientos por unidad Disponibilidad
  Modelo I (x1) Modelo II (x2) Modelo III (x3)  
A 2 3 5 4 000
B 4 2 7 6 000
Demanda minima (und.) 200 200 150 —
Precio por unidad ($) 30 20 50 —
ⓘ Datos adicionales: Las horas de trabajo por unidad del Modelo I son el doble que las del II y el triple que las del III. La capacidad total de la fabrica equivale a 1 500 unidades del Modelo I. El mercado exige producir en proporciones 3 : 2 : 5 para los modelos I, II y III respectivamente.

2Formulacion Matematica

Variables de decision

x1 = unidades producidas del Modelo I
x2 = unidades producidas del Modelo II
x3 = unidades producidas del Modelo III

Funcion objetivo

Maximizar   Z = 30x1 + 20x2 + 50x3

Restricciones

(1) 2x1 + 3x2 + 5x3 ≤ 4 000    (Materia Prima A)
(2) 4x1 + 2x2 + 7x3 ≤ 6 000    (Materia Prima B)
(3) 6x1 + 3x2 + 2x3 ≤ 9 000    (Fuerza de trabajo *)
(4) 2x1 − 3x2 = 0    (Proporcion I : II = 3 : 2)
(5) 5x2 − 2x3 = 0    (Proporcion II : III = 2 : 5)
(6) x1 ≥ 200,   x2 ≥ 200,   x3 ≥ 150    (Demanda minima)
(7) x1, x2, x3 ≥ 0

(*) Las horas por unidad son hI = h, hII = h/2, hIII = h/3. La restriccion h·x1 + (h/2)x2 + (h/3)x3 ≤ 1500h se simplifica multiplicando por 6/h: 6x1 + 3x2 + 2x3 ≤ 9000.

3Reduccion por las Restricciones de Proporcion

Las restricciones (4) y (5) son igualdades. Se introduce el parametro t ≥ 0 tal que:

De (4):   2x1 = 3x2  ⇒  x1 = 3t
De (5):   5x2 = 2x3  ⇒  x3 = 5t
Se fija:   x2 = 2t

Sustituyendo en la funcion objetivo:

Z = 30(3t) + 20(2t) + 50(5t) = 90t + 40t + 250t = 380t

Sustituyendo en las restricciones:

(1') 2(3t) + 3(2t) + 5(5t) = 6t + 6t + 25t = 37t ≤ 4 000  ⇒  t ≤ 108.108...
(2') 4(3t) + 2(2t) + 7(5t) = 12t + 4t + 35t = 51t ≤ 6 000  ⇒  t ≤ 117.647...
(3') 6(3t) + 3(2t) + 2(5t) = 18t + 6t + 10t = 34t ≤ 9 000  ⇒  t ≤ 264.706...
(6a) 3t ≥ 200  ⇒  t ≥ 66.67
(6b) 2t ≥ 200  ⇒  t ≥ 100  ← restriccion activa
(6c) 5t ≥ 150  ⇒  t ≥ 30

La cota inferior activa es t ≥ 100. Para aplicar el Simplex en forma estandar, se hace el cambio de variable:

w = t − 100  ≥  0   ⇒   t = w + 100

Reescribiendo:

Z = 380(w + 100) = 380w + 38 000
Equivalente a maximizar Z' = 380w (y sumar 38 000 al final)
(1'') 37w ≤ 4 000 − 37(100) = 300
(2'') 51w ≤ 6 000 − 51(100) = 900
(3'') 34w ≤ 9 000 − 34(100) = 5 600
(7) w ≥ 0

4Forma Estandar para el Simplex

Se agregan variables de holgura s1, s2, s3 ≥ 0 para convertir las desigualdades en igualdades:

Maximizar   Z' = 380w

(R1) 37w + s1            = 300
(R2) 51w      + s2        = 900
(R3) 34w               + s3 = 5 600
  w, s1, s2, s3 ≥ 0

Solucion basica factible inicial:   w = 0,   s1 = 300,   s2 = 900,   s3 = 5 600,   Z' = 0.

5Metodo Simplex — Iteracion 1

Tableau inicial (Iteracion 0)

Base w s1 s2 s3 RHS Razon (RHS/col w)
s1 37 1 0 0 300 300 / 37 = 8.108 ← min
s2 51 0 1 0 900 900 / 51 = 17.647
s3 34 0 0 1 5 600 5600 / 34 = 164.706
Z' -380 0 0 0 0 —
Criterio de entrada: El coeficiente mas negativo en la fila Z es −380, por lo tanto w entra a la base.
Criterio de salida (razon minima): min{300/37, 900/51, 5600/34} = 300/37 ≈ 8.108, entonces s1 sale de la base.
Elemento pivote: 37 (fila s1, columna w).

Operaciones de pivoteo

Nueva R1 = R1 / 37:
   [37/37   1/37   0   0   |   300/37]
   = [1      1/37   0   0   |   300/37]

Nueva R2 = R2 − 51 × NuevaR1:
   [51−51   0−51/37   1   0   |   900 − 51(300/37)]
   = [0      −51/37    1   0   |   (33 300 − 15 300)/37]
   = [0      −51/37    1   0   |   18 000/37]

Nueva R3 = R3 − 34 × NuevaR1:
   [34−34   0−34/37   0   1   |   5 600 − 34(300/37)]
   = [0      −34/37    0   1   |   (207 200 − 10 200)/37]
   = [0      −34/37    0   1   |   197 000/37]

Nueva FilaZ = FilaZ + 380 × NuevaR1:
   [−380+380   0+380/37   0   0   |   0 + 380(300/37)]
   = [0         380/37     0   0   |   114 000/37]

Tableau final (Iteracion 1)

Base w s1 s2 s3 RHS
s2 1 1/37 0 0 300/37
s2 0 −51/37 1 0 18 000/37
s3 0 −34/37 0 1 197 000/37
Z' 0 380/37 0 0 114 000/37
Criterio de optimalidad: Todos los coeficientes de la fila Z son no negativos (380/37 > 0, y los demas son 0). El tableau es optimo. Se detiene el algoritmo.

6Retrosustitucion — Solucion Original

Paso 1: Obtener w

w = 300/37 ≈ 8.108

Paso 2: Recuperar t

t = w + 100 = 300/37 + 100 = (300 + 3 700)/37 = 4 000/37

Paso 3: Calcular las variables originales

x1 = 3t = 3 × (4 000/37) = 12 000/37 ≈ 324.32 unidades
x2 = 2t = 2 × (4 000/37) = 8 000/37 ≈ 216.22 unidades
x3 = 5t = 5 × (4 000/37) = 20 000/37 ≈ 540.54 unidades

Paso 4: Calcular el valor optimo de Z

Z = 380t + 0 = 380 × (4 000/37) = 1 520 000/37 ≈ $41 081.08

Verificacion directa:
Z = 30(12 000/37) + 20(8 000/37) + 50(20 000/37)
   = (360 000 + 160 000 + 1 000 000) / 37 = 1 520 000 / 37 ≈ $41 081.08

7Verificacion de Restricciones

Con x1 = 12000/37, x2 = 8000/37, x3 = 20000/37:

Restriccion Calculo Limite Estado
Materia Prima A:   2x1 + 3x2 + 5x3 2(12000/37) + 3(8000/37) + 5(20000/37) = 148000/37 = 4000 ≤ 4 000 ✓ ACTIVA
Materia Prima B:   4x1 + 2x2 + 7x3 4(12000/37) + 2(8000/37) + 7(20000/37) = 204000/37 ≈ 5513.5 ≤ 6 000 ✓ holgura = 486.5
Fuerza de trabajo:   6x1 + 3x2 + 2x3 6(12000/37) + 3(8000/37) + 2(20000/37) = 136000/37 ≈ 3675.7 ≤ 9 000 ✓ holgura = 5324.3
Proporcion I:II   2x1 − 3x2 2(12000/37) − 3(8000/37) = (24000 − 24000)/37 = 0 = 0 ✓
Proporcion II:III   5x2 − 2x3 5(8000/37) − 2(20000/37) = (40000 − 40000)/37 = 0 = 0 ✓
Demanda minima x1 12000/37 ≈ 324.3 ≥ 200 ✓
Demanda minima x2 8000/37 ≈ 216.2 ≥ 200 ✓
Demanda minima x3 20000/37 ≈ 540.5 ≥ 150 ✓

La restriccion activa (saturada) es la de Materia Prima A, que es el recurso escaso que limita la produccion optima.
